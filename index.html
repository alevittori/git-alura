Codigcos para saber quien sos para git

Antes de cualquier interacción con git, es necesario informar quién eres para que almacene correctamente los datos del autor de cada uno de los cambios en el código.

git config --local user.name "Tu nombre aquí" git config --local user.email "Tu@email.aqui”

COmando para ver la lista de commit que realizamos

git log git log --oneline git log -p
